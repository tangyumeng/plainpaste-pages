SnapTitle Support

SnapTitle watches your screenshots folder and renames new captures with on-device OCR.

Contact

Email: tangyumeng@gmail.com

When reporting an issue

  • macOS version
  • SnapTitle version and build number
  • Watch folder path (Desktop / Screenshots / custom)
  • What happened and what you expected

How to use

  1. Open SnapTitle from the menu bar (text.viewfinder icon).
  2. Choose your Screenshots or Desktop folder.
  3. Take a screenshot (⌘⇧3 / ⌘⇧4). The file renames shortly after.
  4. Use Undo from the menu if needed.
