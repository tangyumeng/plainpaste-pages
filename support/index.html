When reporting an issue

Please include:

  • macOS version
  • PlainPaste version and build number
  • The action you used (Paste as Plain Text or Replace Clipboard with Plain Text)
  • The app where you tried to paste
  • What happened and what you expected to happen

Accessibility permission

Automatic paste needs macOS Accessibility permission to send Command + V to the frontmost app. If PlainPaste is not listed in System Settings, click + in Privacy & Security > Accessibility and choose PlainPaste.app. Replace Clipboard with Plain Text still works without this permission.
